# docs/

Engineering documentation that lives with the code — build/packaging conventions, operational
notes, and per-subsystem engineering docs as they accrete. **Design authority does not live
here**: the normative requirements, architecture, roadmap, and decision locks (L-1…L-59) are
the owner's Context Engine design records; documents in this folder record how this repository
implements them (e.g. `versioned-install.md` for R-VER-004).
